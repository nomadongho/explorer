# explorer — Leo's Learning Apps Homepage

A polished, mobile-friendly landing page for [Leo's Explorer](https://nomadongho.github.io/) — a small collection of interactive learning apps made for Leo Park.

## Apps

| App | URL |
|-----|-----|
| Math Explorer | https://nomadongho.github.io/math-explorer |
| English Explorer | https://nomadongho.github.io/english-explorer |
| Clock Explorer | https://nomadongho.github.io/clock-explorer |

## Files

| File | Purpose |
|------|---------|
| `index.html` | Main landing page |
| `style.css` | All styles — responsive, mobile-first |
| `script.js` | Minimal JS: footer year + scroll animations |

## Deployment

This site is designed for **GitHub Pages** with no build step required.

1. Place `index.html`, `style.css`, and `script.js` in the root of the `nomadongho/explorer` repository.
2. Enable GitHub Pages in **Settings → Pages → Deploy from branch → main → / (root)**.
3. Visit `https://nomadongho.github.io/` to see the live page.

## Tech

- Plain HTML, CSS, and JavaScript — no frameworks or build tools
- Google Fonts (Nunito + Lora) loaded from CDN
- Responsive down to 320 px
- Respects `prefers-reduced-motion`
